# KMS 知识管理系统 - 页面设计文档

> 创建日期: 2026-04-02
> 设计风格: 现代卡片型（绿色系、圆角卡片、模块化布局）

---

## 一、项目概述

### 1.1 项目定位
通用知识管理平台，支持知识存储、智能检索、AI辅助写作、知识运营等核心能力。

### 1.2 用户角色
- **管理员**: 拥有所有功能权限，可管理知识库、用户、系统配置
- **普通用户**: 可使用AI对话、智能搜索、智能写作等核心功能

### 1.3 核心能力
- AI博士综合助手（问答+创作+分析）
- 智能+语义混合检索
- 文档解析流水线
- 语料标注工作台
- 知识增强管理
- 知识运营飞轮
- 前置模板库
- 知识图谱可视化
- 数据统计分析

## 三、页面详细设计

### 3.1 AI博士对话

**布局**: 左右两栏

**功能模块**:
- 顶部工具栏：新对话、历史记录、导出
- 模式切换Tab：问答模式 / 创作模式 / 分析模式
- 对话消息区：
  - AI消息（含溯源卡片）
  - 用户消息
- 输入区：附件、语音、@知识库、#技能
- 右侧功能面板：
  - 当前会话信息
  - 检索范围选择
  - 真实性验证状态
  - 快捷技能按钮

**溯源卡片结构**:
```
📚 引用来源
├── 知识库：公共政策库 / 国家标准规范
├── 文档：2024数字化转型指南.pdf (第12-15段)
└── 置信度：98% ✅ 已验证
```

**新增能力**:
- 三种模式切换
- 溯源文档卡片
- 真实性验证状态面板
- 快捷技能按钮
- 历史记录、导出功能

---

### 3.2 智能搜索

**布局**: 顶部搜索栏 + 结果列表

**功能模块**:
- 搜索框（支持混合/语义/关键词三种模式）
- 筛选条件：文档类型、时间范围、知识库、高级筛选
- 结果统计：总数、用时、排序方式
- 结果卡片：
  - 文件类型图标
  - 标题（高亮关键词）
  - 摘要片段
  - 元数据（日期、知识库、匹配度）
  - 操作按钮（收藏、复制）
- 分页导航

**新增能力**:
- 三种搜索模式切换
- 高级筛选面板
- 时间范围选择器
- 排序方式切换
- 导出搜索结果
- 收藏功能

---

### 3.3 智能写作

**布局**: 三栏布局

**功能模块**:
- 左侧面板：
  - 写作模板选择
  - 任务配置（发文单位、主题、时间等）
  - 参考知识库选择
- 中间编辑区：
  - 编辑器工具栏
  - 文档预览（公文格式）
  - AI辅助工具栏（续写、润色、校验、参考）
- 右侧面板：
  - 历史公文参考推荐
  - 格式校验状态
  - AI生成记录

**新增能力**:
- 任务配置面板
- 参考知识库选择
- 历史公文参考推荐
- 格式校验状态实时显示
- AI辅助工具栏
- 保存草稿功能

---

### 3.4 知识库管理

**布局**: 三栏布局

**功能模块**:
- 左侧：知识分层导航树
- 中间：文件列表表格
  - 解析状态列（待解析/解析中/已解析）
  - 版本列
  - 批量操作栏
- 右侧：元数据配置面板
  - 分类标注
  - 标签管理
  - 版本历史
  - 解析详情

**新增能力**:
- 视图切换（文件库/解析流水线）
- 批量操作功能
- 解析状态显示
- 版本管理
- 解析详情展示

---

### 3.5 文档解析流水线

**布局**: 左右两栏

**功能模块**:
- 顶部统计卡片：待处理、处理中、已完成、失败
- 左侧流程图概览：
  - 5节点可视化（上传→解析→分块→向量化→入库）
  - 当前进度线
  - 当前处理文档
  - 资源监控（CPU/内存/队列）
- 右侧任务列表：
  - 状态筛选
  - 任务卡片（分阶段进度条）
  - 任务控制（暂停/取消/重试）
  - 错误详情

**配置选项**:
- 解析引擎：MinerU / PyMuPDF / Paddle-OCR
- 分块策略：智能分块 / 固定大小 / 段落分块
- 向量化模型：BGE-large / text-embedding / 自定义

---

### 3.6 语料标注工作台

**布局**: 三栏布局

**功能模块**:
- 左侧任务队列：
  - 状态统计（待审核/已通过/已驳回）
  - 任务列表
  - 批量操作
- 中间标注区：
  - 文档预览（标注高亮）
  - 标注工具栏
- 右侧结果面板：
  - 分类标注
  - 实体标注（地区/时间/数值/组织）
  - 关键词标签
  - 审核操作

**智能预标注规则**:
- 自动分类规则（标题关键词、文号格式、发文单位）
- 实体抽取规则（地区、时间、数值、组织）
- 关键词抽取（TF-IDF、TextRank、关键词库匹配）

### 3.7 知识增强管理

**布局**: 三栏布局

**功能模块**:

  - 任务统计（QA/泛化/拆解）
  - 来源文档列表

  - 泛化处理：同义问法管理
  - 拆解处理：知识单元拆解

- 右侧配置统计：

  - 质量统计指标

### 3.8 知识运营飞轮

**布局**: 顶部流程图 + 下方两栏

**功能模块**:

  用户反馈 → 质量评估 → 知识补全 → 规则优化 → 循环
- 左侧反馈收集：
  - 反馈分类（好评/差评/纠错）
  - 反馈处理（补充知识/修正/标记）
- 右侧运营看板：
  - MetaPrompt编排（版本管理）
  - 质量指标看板（准确率/覆盖率/满意度/时效）
  - 知识缺口识别（优先级排序）
